## Problem Statement

Problem #56 - Merge Intervals

Difficulty: Medium

## Approaches

### Sort and Merge

- Time Complexity: O(n log n)
- Space Complexity: O(n)
- Sort intervals by start time, then merge overlapping ones.

### Heap / Priority Queue

- Time Complexity: O(n log n)
- Space Complexity: O(n)
- Use a min-heap to track and merge intervals.
